# 2.1 深度学习基本理论

## 人工神经网络与万能近似定理

### 人工神经网络（ANN）

人工神经网络受生物神经系统启发，由大量相互连接的"神经元"组成。每个神经元接收输入信号，经过加权求和与非线性变换后产生输出。

### 万能近似定理（Universal Approximation Theorem）

**定理内容**：一个包含足够多神经元的单隐层前馈神经网络，可以以任意精度逼近任何连续函数。

这意味着神经网络具有极强的函数拟合能力，理论上可以学习任意复杂的映射关系。在EDA领域，这意味着我们可以用神经网络来近似芯片设计中的各种物理模型和优化函数。

## 神经元的基本结构与功能

一个神经元的计算过程可以表示为：

```
输入: x1, x2, ..., xn
权重: w1, w2, ..., wn
偏置: b

线性组合: z = w1*x1 + w2*x2 + ... + wn*xn + b
激活输出: a = f(z)
```

用矩阵形式表示：

```
z = W * x + b
a = f(z)
```

其中 W 是权重矩阵，x 是输入向量，b 是偏置向量，f 是激活函数。

## 常见的激活函数及其特点

激活函数为神经网络引入非线性，使其能够学习复杂的模式。

### Sigmoid

```
f(x) = 1 / (1 + exp(-x))
```

- 输出范围：(0, 1)
- 优点：输出可解释为概率
- 缺点：梯度消失问题、输出非零中心

### Tanh

```
f(x) = (exp(x) - exp(-x)) / (exp(x) + exp(-x))
```

- 输出范围：(-1, 1)
- 优点：零中心输出
- 缺点：仍有梯度消失问题

### ReLU（Rectified Linear Unit）

```
f(x) = max(0, x)
```

- 优点：计算简单、缓解梯度消失、稀疏激活
- 缺点：Dead ReLU问题（负区间梯度为0）
- **目前最常用的激活函数**

### Leaky ReLU

```
f(x) = max(0.01*x, x)
```

- 解决Dead ReLU问题

### GELU（Gaussian Error Linear Unit）

```
f(x) = x * Phi(x)
```

- Transformer中常用的激活函数
- 输出平滑，效果优于ReLU

## 前向传播与反向传播

### 前向传播（Forward Propagation）

数据从输入层逐层传递到输出层的过程：

```
输入层 -> 隐藏层1 -> 隐藏层2 -> ... -> 输出层

z[1] = W[1] * x + b[1]
a[1] = f(z[1])
z[2] = W[2] * a[1] + b[2]
a[2] = f(z[2])
...
y_hat = a[L]
```

### 损失函数（Loss Function）

衡量预测值与真实值的差距：

- **均方误差（MSE）**：`L = (1/n) * sum((y - y_hat)^2)`，常用于回归任务
- **交叉熵（Cross-Entropy）**：`L = -sum(y * log(y_hat))`，常用于分类任务

### 反向传播（Backpropagation）

利用链式法则计算损失函数对每个参数的梯度：

```
dL/dW[L] = dL/da[L] * da[L]/dz[L] * dz[L]/dW[L]

dL/dW[L-1] = dL/da[L] * da[L]/dz[L] * dz[L]/da[L-1] * da[L-1]/dz[L-1] * dz[L-1]/dW[L-1]
```

### 梯度下降（Gradient Descent）

根据梯度更新参数：

```
W = W - learning_rate * dL/dW
```

常见变体：
- **SGD**（随机梯度下降）：每次用一个样本更新
- **Mini-batch SGD**：每次用一小批样本更新
- **Adam**：自适应学习率，最常用

## 深度神经网络

"深度"指的是网络有多层隐藏层。深层网络可以学习层次化的特征表示：

```
第1层：学习简单特征（边缘、角点）
第2层：学习组合特征（纹理、形状）
第3层：学习高级特征（物体部件）
...
```

### 深度网络的训练挑战

1. **梯度消失/爆炸**：层数增多导致梯度传播困难
2. **过拟合**：参数过多容易记住训练数据
3. **计算开销**：深层网络需要更多计算资源

### 解决方案

- **Batch Normalization**：归一化中间层输出
- **Dropout**：随机丢弃部分神经元
- **残差连接（Residual Connection）**：ResNet的核心创新
- **权重初始化**：Xavier/He初始化
