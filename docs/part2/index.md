# 第二部分：AI4EDA——深度学习方法、Pytorch编程和TCAD仿真加速

## 课程概览

本部分介绍深度学习的基本理论和PyTorch编程框架，并通过TCAD仿真加速的案例展示AI在EDA领域的初步应用。

## 学习目标

- 掌握深度学习的基本理论：神经网络、激活函数、前向/反向传播
- 了解常见神经网络类型：FNN、CNN、RNN、GNN
- 熟练使用PyTorch搭建和训练神经网络
- 理解深度学习用于TCAD器件建模的原理和方法
