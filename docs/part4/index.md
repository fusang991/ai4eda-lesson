# 第四部分：AI4EDA——APR CTS时钟树综合

## 课程概览

本部分介绍芯片物理设计中的时钟树综合（CTS）过程和相关算法，以及GAN、图神经网络在时钟树预测和优化中的应用。

## 学习目标

- 理解片上时钟系统和时钟树的基本概念
- 掌握CTS的完整过程：Cluster和Balance
- 理解CTS相关算法：聚类、DME、Steiner树
- 了解GAN和GNN在CTS中的应用
