## 课程简介

本课程系统介绍深度学习在集成电路设计自动化（EDA）领域的应用。从Python编程基础和深度学习理论出发，逐步深入到芯片物理设计（APR）的各个阶段，包括布局（Placement）、时钟树综合（CTS）和绕线（Routing），并结合Google AlphaChip、DREAMPlace等经典工作和EDA设计精英挑战赛的实战赛题，帮助学员全面掌握AI4EDA的核心方法。

## 学习目标

1. **掌握Python编程和深度学习方法** — PyTorch框架、图神经网络、强化学习等
2. **熟悉EDA全流程** — 从RTL到GDSII，了解各阶段算法原理
3. **深入理解AI在APR中的应用** — 布局、时钟树、绕线中的AI方法
4. **具备AI建模能力** — 通过实战赛题积累AI4EDA项目经验
