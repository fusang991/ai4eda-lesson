# 第五部分：AI4EDA——Route绕线、逻辑综合与电源完整性

## 课程概览

本部分是课程的最后一个模块，涵盖芯片布线（Routing）、逻辑综合（Logic Synthesis）、电源完整性（IR Drop）和Transformer模型等内容。

## 学习目标

- 理解芯片布线的完整过程和相关算法
- 了解深度学习在布线中的应用
- 掌握IR Drop预测的深度学习方法
- 理解逻辑综合的基本概念和AI方法
- 掌握Transformer模型的原理
