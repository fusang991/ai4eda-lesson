# 第一部分：EDA集成电路设计自动化和Python编程

## 课程概览

本部分是整个课程的基础模块，主要介绍EDA设计自动化的基本概念、芯片设计的完整流程，以及Python编程基础。通过本部分的学习，学员将建立起对EDA领域的整体认识，并掌握后续课程所需的Python编程技能。

## 学习目标

- 理解EDA设计自动化的基本概念和AI4EDA的意义
- 掌握芯片设计从Spec到Tape Out的完整流程
- 熟练使用Python进行编程，包括NumPy、Matplotlib、SciPy等常用库
- 理解并实现超图分割算法（KL/FM算法）
